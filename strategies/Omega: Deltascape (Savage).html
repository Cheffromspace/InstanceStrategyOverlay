Omega: Deltascape (Savage):
- Main 8-player raid in Stormblood, released on July 18, 2017
- Requires item level 310+ to enter
- Access via Duty Finder → Raids → Savage Raids (Stormblood)
- Can enter solo or with <8 players using Unrestricted Party
- Unlock: Complete quest "The Anomaly", then interact with Magitek Terminal in Rhalgr's Reach (x13.5, y12.4)

Floors:
- Deltascape V1.0 (Savage): Boss Alte Roite
- Deltascape V2.0 (Savage): Boss Catastrophe
- Deltascape V3.0 (Savage): Boss Halicarnassus
- Deltascape V4.0 (Savage): Bosses Exdeath, Neo Exdeath

Item level restrictions do not apply for full/unrestricted parties.
